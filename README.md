# Lab Part 1 - Suppliers & Products CRUD

## Công nghệ sử dụng
- Node.js + Express
- MongoDB (Mongoose)
- EJS + Bootstrap
- Swagger cho API Docs

## Cách chạy
1. Copy `.env.example` → `.env`
2. Chỉnh `MONGODB_URI`
3. `npm install`
4. `npm run dev`
5. Mở [http://localhost:3000](http://localhost:3000)
6. Swagger: [http://localhost:3000/api-docs](http://localhost:3000/api-docs)
